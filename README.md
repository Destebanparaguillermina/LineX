# LineX
**¿Que es LineX?**

**• R= LineX es una herramienta para obtener datos de un numero de telefono utilizando la API de Numverify**

**• Con soporte para 232 Paises**

**• Instalacion y uso:**
```
# Instalamos Git

apt install git -y
```
```
# Instalamos Python

apt install python -y
```
```
# Instalamos Python3

apt install python3 -y
```
```
# Clonamos el repositorio

git clone https://github.com/Euronymou5/LineX.git
```
```
# Instalamos el modulo requests

pip install requests
```
```
# Iniciamos la herramienta

(Linux / Termux): python3 linex.py   

(Windows):  python linex_win.py
```
